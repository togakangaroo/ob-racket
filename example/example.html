1 Example Org Babel File With Racket

This is an example Org file showing some suggested ways to use the ob-racket module with Org Babel integration for Racket. To see the raw source for this file (instead of Github's processed version), tap the Raw button at the top right of this frame.

1.1 Recursive Factorial

1.1.1 Definition

The code block below shows the recursive factorial that everyone knows and loves: 

(define (factorial n)
  (if (= n 1)
      1
      (* n (factorial (sub1 n)))))

I set the following header arguments for this block:

:tangle
Set to yes, so that the block will be included in example.rkt when we tangle this file.
:eval
Set to no. This block only contains a function definition - there's no point in executing it by itself with no input.
:exports
Set to code, so only the code will be exported.

1.1.2 Execution

The following block sets up a way to execute the factorial function, so it can be quickly called with arbitrary input using the #+CALL macro. We use the noweb syntax to include the definition of factorial from the define-factorial block above. The code block looks like this: 

#+NAME: factorial
#+BEGIN_SRC racket :tangle no :noweb yes :exports code
<<define-factorial>>
(factorial input)
#+END_SRC

And the expanded output code block looks like this: 

(define (factorial n)
  (if (= n 1)
      1
      (* n (factorial (sub1 n)))))
(factorial input)

Header arguments are as follows:

:tangle
Set to no. The definition is already being tangled above, we don't need to tangle it again.
:noweb
Enables the noweb syntax we use to embed the function definition.
:exports
In this example I have set this to code, so you can see the noweb syntax. Normally you would just set this to none to avoid cluttering up the output file.

1.1.3 Calling

Once the above is set up, you can quickly call the factorial function using a #+CALL macro, like the following: 

#+CALL: factorial(input=10)

For example, the factorial of 10 is: 

3628800
